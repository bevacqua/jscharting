The chart is able to detect invalid properties and settings and logs them in the browser debug console.
This sample intentionally uses incorrect settings to demonstrate this feature.
Notice the tiny x mark at the top left of the chart. This indicates there was an issue with some given settings in the chart JSON.
This feature can also be disabled by setting the chart JSON debug property to false.
This information can also be retrieved programatically to detemine whether an issue has occured.
