Your First Chart

To begin, create a new HTML file. This file can be created within the JSC bundle or built off of a sample in /Samples. In order to use the charts in any other location, copy the JSC folder from within the bundle to your working directory. This is the minimum requirement to work with JSCharting. First, include jQuery and JSCharting libraries in your HTML file as follows: <script type="text/javascript" src="JSC/jquery-latest.min.js"></script> <script type="text/javascript" src="JSC/JSCharting.js"></script> jQuery version 1.10.1 is shown in this example, however, the version shipped with the bundle should be used.

In the body of the HTML page, add a div element with a specific ID attribute to contain the chart graphics. <div id="chartDiv" style="width: 640px; height: 400px"></div> Now add a script block to define the chart with a JSON configuration to initialize and render the chart in the div#chartDiv tag. The div ID is specified through the targetElement property. <script type="text/javascript">

var config = {
targetElement: 'chartDiv', series: [
{
name: 'Purchases',
points: [
[new Date(2010, 0, 1),29.9]
]
}]
};

$(document).ready(function () {
/*Now initialize the chart with this config setting and a chart will render in place of the div tag with the 'chartDiv' ID.*/ var chart = new JSC.Chart(config);
});

</script> Open this page in your web browser to view your newly configured chart! Note: It is important to be aware of the chart's Code Expansion feature when looking at sample code. In order to specify the chart types effectively, it is also important to understand Enum Chaining feature. More on using chart types is available in the chart types tutorials.
