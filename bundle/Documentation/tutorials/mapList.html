Map Codes

JSMapping uses standard two letter country and state province codes so getting a map of the United States (US) or Australia (AU) can be done with a single setting and a more detailed map of a given state or province like California (US.CA) or Tasmania (AU.TS) is just as easy to highlight or bind your own data. See the full countries list and state/province list supported for all supported options.

Regions

Specific maps and areas are specified by naming one of the following regions:

  • Americas
  • Europe
  • Asia
  • Africa
  • Australia

Countries

By using two letter ISO codes, or hasc codes.

State/Province

(HASC codes start with two letter ISO for country, then generally two letter codes for states or provinces such as US.IL for Illinois USA.). Some edge cases use hasc codes with three levels for provinces.
