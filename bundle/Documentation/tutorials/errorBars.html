Introduction

A number of series types also support error bars. Error bars are associated with points and defined under the subvalue point property.

Types.subvalue

Example error bars

Figure 1. Samples.ColumnError

Setting Values

The subvalue.range property can be set as an array of high and low values as shown below.

{ series:[{ points:[{name:"Nolan",y:16,subvalue:{range:[15,18]}] }] } Samples.timeLineError

A numeric offset (subvalue.offset) can also be used to specify the range as the point's y value + and - the offset.

{ series:[{ points:[{name:"Nolan",y:16,subvalue:{offset:5}] }] } Samples.lineErrorOffset

A percentage offset (subvalue.percent) can also be used and indicates the error range is + and - that percentage of the point's y value.

{ series:[{ points:[{name:"Nolan",y:16,subvalue:{percent:5}] }] } Samples.lineErrorStyling Subvalue percentage offsets can be applied to chart's defaultPoint and does not have to be repeated for each data point.
